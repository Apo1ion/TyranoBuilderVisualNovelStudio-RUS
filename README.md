# TyranoBuilderVisualNovelStudio-RUS

![Image alt](https://github.com/Apo1ion/TyranoBuilderVisualNovelStudio-RUS/blob/main/Logo-TyranoBuilderVisualNovelStudio-RUS.png)

Русификация для TyranoBuilder Visual Novel Studio.

Данный русификатор предназначен для актуальной версии приложения TyranoBuilder. На момент 10.03.2026 перевод подготовлен для версии программы V3.06.

Русификатор может быть установлен на официальную версию программы, скачанную через Steam.
Установка:

  1) Перейдите в папку, где установлена программа TyranoBuilder. (Диск:\Steam\SteamApps\common\TyranoBuilder)
  2) В директории программы создайте папку с названием lang, если она отсутствует.
  3) Поместите в эту папку файл en.json, скачанный с GitHub.

Если программа была установлена через Steam, папка lang обычно уже существует. В этом случае достаточно просто поместить файл en.json в эту папку.

Обратите внимание: в папке lang уже может находиться файл en.json, который отвечает за английский перевод интерфейса. После замены этого файла на скачанный с GitHub, английский перевод будет заменён русским.

В итоге путь к файлу должен выглядеть следующим образом:
Диск:\Steam\SteamApps\common\TyranoBuilder\lang\en.json

Russian localization for TyranoBuilder.

This Russian localization is intended for the current version of TyranoBuilder. As of 10.03.2026, the translation is prepared for version V3.06 of the program.

This localization can be installed on the official version of the program downloaded through Steam.
Installation:

1) Go to the folder where TyranoBuilder is installed. (Disk:\Steam\SteamApps\common\TyranoBuilder)
2) In the program directory, create a folder named lang if it does not already exist.
3) Place the en.json file downloaded from GitHub into this folder.

If the program was installed through Steam, the lang folder usually already exists. In this case, you only need to place the en.json file into this folder.

Please note: the lang folder may already contain an en.json file that provides the English interface translation. After replacing this file with the one downloaded from GitHub, the English translation will be replaced with the Russian localization.

The final file path should look like this:
Disk:\Steam\SteamApps\common\TyranoBuilder\lang\en.json
